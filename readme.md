## Pre-requisitos

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en/)

## Instalación

```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/comandos-gits.git

# Instalar dependencias
npm install
```

## Uso

```bash
# Iniciar la aplicación
npm start
```
